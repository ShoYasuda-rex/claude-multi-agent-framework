
# Get Web Sounds

フリーのBGM・SE（効果音）を調査→比較提示→ダウンロード→配置→クレジット管理まで自動化するスキル。
ゲーム・Webアプリ・ランディングページなど用途を問わず使える。

> **使い分け**:
> `/asset` → 音源 — フリーBGM・SEを調達（ライセンス確認付き）
> `/asset` → 画像 — ドット絵をローカルAIでバッチ生成
> 画像素材 — itch.io等で手動調達、またはブラウザ版AI（ChatGPT, Grok等）で生成

---

## 0. プロジェクト分析と推奨提案

プロジェクトを分析し、不足しているBGM・SEを特定して推奨を提案する。

### 0-1. プロジェクト情報の収集（並列実行）

以下を並列で実行:

- CLAUDE.md を Read — 技術スタック・テーマ・用途を把握
- `docs/CORE.md` または `docs/ARCHITECTURE.md` を Read — コンセプト・ターゲットユーザーを把握（なければスキップ）
- `public/assets/` と `src/assets/` を Glob — 既存アセットの棚卸し
- `src/` 配下で音声の import/参照を Grep — コードが参照しているアセットパスを特定
- `CREDITS.md` を Read — 既に取得済みのアセットを把握（なければスキップ）

### 0-2. ギャップ分析

収集した情報から以下を判定:

- **プロジェクトの種別・用途**（Webアプリ、ゲーム、LP、業務ツール等）
- **既存BGM・SEの状況**（何がある / 何が足りない）
- **コードが参照しているが存在しない音声ファイル**（404候補）
- **雰囲気・トーン**（既存BGMのジャンルや世界観）

### 0-3. 推奨提案

分析結果をもとに、具体的な推奨を提示する:

```
## プロジェクト分析結果

- 種別: {判定結果}（Webアプリ / ゲーム / LP 等）
- テーマ: {雰囲気・トーン}
- 既存アセット: SE {N}件 / BGM {N}件
- スタイル: {既存アセットの特徴}

## 推奨アセット

| # | 種別 | 推奨内容 | 理由 |
|---|------|---------|------|
| 1 | {SE/BGM} | {具体的な内容} | {なぜ必要か} |
| 2 | {SE/BGM} | {具体的な内容} | {なぜ必要か} |
| 3 | {SE/BGM} | {具体的な内容} | {なぜ必要か} |
```

AskUserQuestion で確認:

**質問: どのアセットを取得しますか？**（multiSelect: true）
- 推奨候補を選択肢として提示（上位3-4件）
- 各選択肢の description に理由を記載

ユーザーの選択を受けて、**選択された全アセットを一括で調査→取得→配置する**（場面ごとの個別確認はしない）。

## 1. 配置先パスの決定

ステップ 0 で種別が決まった状態で:

1. ステップ 0 で取得した既存アセットのディレクトリ構造を参照
2. **既存構造があればそれに従う**（例: `public/assets/se/` が既にあればそこに配置）
3. **既存構造がなければ `public/assets/` に配置**（SE → `se/`、BGM → `bgm/`）
4. ディレクトリが存在しなければ `mkdir -p` で作成

---

## 2. 一括調査・選定

### 調査ソース

#### SE（効果音）
1. **効果音ラボ** — 日本語、商用OK、クレジット不要、ログイン不要
   - URL: `https://soundeffect-lab.info/sound/anime/` 等カテゴリページから探す
2. **魔王魂** — 日本語、商用OK、会員登録不要
   - URL: `https://maou.audio/category/se/`

#### BGM
1. **魔王魂** — 日本語、商用OK、会員登録不要
   - URL: `https://maou.audio/category/bgm/`
2. **DOVA-SYNDROME** — 日本語、商用OK、ログイン不要
   - URL: `https://dova-s.jp/_contents/search/?keyword={query}`

#### 共通
- WebSearch も併用して最新のアセットを探す
- **WebSearch経由のDL制限**: 上記の指定サイト以外のURLは手動DL案内に切り替える
- 日本語サイトは日本語クエリで検索
- **同作者優先**: CREDITS.md に既登録の作者を優先（スタイル統一に有効）

### 選定ルール

各アセットについて **最適な1件を自動選定** する。場面ごとにユーザーに聞かない。

選定基準（優先順）:
1. **ライセンス**: CC0 > クレジット不要独自 > CC-BY > CC-BY-SA。不明・GPLはスキップ
2. **信頼性**: DL数・評価が高い、作者の実績がある
3. **同作者優先**: CREDITS.md 既登録の作者があれば優先
4. **形式**: SE: WAV > OGG > MP3、BGM: OGG > MP3 > WAV

---

## 3. 取得・配置

1. **直接DL可能か確認**
   - 効果音ラボ: 直接DLリンクを取得
   - 魔王魂: DLリンクを取得
   - DOVA-SYNDROME: DLリンクを取得
   - **直接DL不可** → URLを提示し、手動DLを案内して終了

2. **ダウンロード実行**
   - `curl -L -o {filename} {url}` でダウンロード

3. **配置**
   - ステップ 1 で決定したパスにファイルを配置
   - 必要に応じてリネーム（スネークケース、用途がわかる名前）

4. **形式変換（必要な場合のみ）**
   - `ffmpeg -i {input} {output}` （FFmpeg）
   - 例: WAV→OGG、MP3→OGG
   - FFmpeg 未インストール時は手動変換を案内

5. **確認**
   - 配置したファイルを `ls -la` で確認

## 6. コード適用

配置したBGM・SEを実際にコードに組み込む。

### 6-1. 適用先の特定

ステップ 0 の分析結果を活用:

- **コードが参照しているが存在しなかった音声ファイル** → パスを合わせて配置済みなら適用不要
- **新規追加のアセット** → 使用箇所を特定または新規作成

Grep で既存の音声再生コードのパターンを検索:
- `new Audio(`, `Audio(`, `.play()`, `howl`, `Howl`, `useSound`, `import.*\.(mp3|ogg|wav)`
- HTML: `<audio`, `<source`

### 6-2. 適用パターン

プロジェクトの技術スタックに合わせて適用:

**バニラJS / HTML:**
```js
const se = new Audio('assets/se/click.mp3');
se.play();
```

**既存の音声管理がある場合:**
- 既存パターンに従って追加（サウンドマネージャー、定数ファイル等）

**BGMのループ再生:**
```js
const bgm = new Audio('assets/bgm/main_theme.mp3');
bgm.loop = true;
bgm.volume = 0.5;
bgm.play();
```

### 6-3. 適用ルール

- **既存の音声再生パターンがあれば必ずそれに従う**（独自実装しない）
- **音声ファイルパスは配置先と一致させる**（相対パス / 絶対パスはプロジェクトの慣習に合わせる）
- **ボリューム設定**: BGMはデフォルト0.3〜0.5、SEはデフォルト0.5〜0.8（既存設定があればそちらに従う）
- **ユーザーインタラクション起因**: ブラウザの自動再生ポリシーに注意。BGMはユーザー操作後に再生開始する

## 信頼性チェック

候補選定時に以下を確認し、信頼性が低いものは候補から除外する:

- **DL数・評価** — ダウンロード数や評価が高いアセットを優先（実績 = 信頼性）
- **作者の実績** — 複数作品を公開している作者を優先
- **同作者・同パック優先** — CREDITS.md に既登録の作者があれば、同じ作者の別アセットを優先候補にする（スタイル統一・ライセンス確認の手間削減）

## 制約事項

- **ログイン不要のソースが基本** — ログインが必要なサイトはスキップ
- **直接DLできないURLはスキップ** — 手動DL案内に切り替え
- **ライセンス不明はスキップ** — 候補から除外し理由を表示
- **商用利用不可ライセンスは警告** — NC付きは明示的に警告
- **ファイルサイズ制限** — 単一ファイル10MB超は警告
- **形式の優先順位** — SE: WAV > OGG > MP3、BGM: OGG > MP3 > WAV

## 禁止事項

- 著作権が不明なアセットのダウンロード
- ライセンス条件を無視したクレジット省略
- 有料アセットの無断使用
